# FShare - User Clients

## Setup React.js app

Install dependency:

```
npm install
```

Run app:

```
npm start
```

Build app:
```
npm run build
```

## Setup Android and iOS app

### Copying Web Code

```
npx cap copy
```

### Building Native Project on iOS
```
npx cap copy ios
npx cap open ios
```

### Building Native Project on Android
```
npx cap copy android
npx cap open android
```
